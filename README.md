### BoundScrollView
类似于ios，带回弹效果的ScrollView

### 使用
Gradle
```
compile 'com.rairmmd:boundscrollview:1.0.0'
```

### 布局中
和ScrollView一样
```
 <com.rairmmd.boundscrollview.BoundScrollView
        android:layout_width="match_parent"
        android:layout_height="match_parent">

    </com.rairmmd.boundscrollview.BoundScrollView>
```
